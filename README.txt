AppNutri Mobile

Requisitos:
	- NodeJS com NPM (ou SUDO NPM para o Linux)

---------------------------------

Execução do app:

- Clone o repositório no seu computador

- Entre na pasta do projeto
	cd appnutri/

- Instale as dependencias do projeto
	npm install

- Inicie o servidor NodeJS
	npm start

Para visualizar o projeto acesse "localhost:3001" no navegador
